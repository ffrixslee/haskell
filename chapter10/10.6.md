1. Write a function that filters for DbDate values and returns a list of the UTCTime values inside them:

```hs
filterDbDate :: [DatabaseItem] -> [UTCTime]
filterDbDate xs = foldr f [] xs
    where f (DbDate t) ts = t : ts
          f _          ts = ts
```
